# Backend

## Setup

1. Go to project folder

> cd Backend

2. Install Dependencies

> npm run setup

3. Create .env with following content

````
DB_URL=

PORT=3000
SECRET=AAA
````

## Run

1. Start Server

> npm start

2. Open in url or using an Endpoint tester (postman)

> http://127.0.0.1:3000/
